# xslt-latex
## XSLT to convert to LaTeX

These are a set of three XSLT files containing templates to transform TEI-XML documents into LaTeX.

Created by [Nicolas Vaughan](http:nicolasvaughan.org) (n.vaughan@oxon.org) in 2020.
